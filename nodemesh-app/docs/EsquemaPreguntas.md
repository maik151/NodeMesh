### La Matriz Maestra: Taxonomía de NodeMesh

| :--- | :--- | :--- | :--- | :--- |
| **`single_choice`** | ⚡ **Local** | Recordar | Aislar una verdad absoluta entre distractores. | **Q:** ¿Estructura por defecto en PostgreSQL para un índice?<br>**Opciones:** A) Hash, B) B-Tree, C) Red-Black, D) LSM.<br>**Solución:** B.<br>**Justificación:** B-Trees permiten búsquedas de rango eficientes ($O(\log n)$). |
| **`multi_choice`** | ⚡ **Local** | Comprender | Exige conocer el panorama completo; cero suerte. | **Q:** Métodos HTTP matemáticamente idempotentes (RFC 7231).<br>**Opciones:** GET, POST, PUT, DELETE, HEAD.<br>**Solución:** GET, PUT, DELETE, HEAD.<br>**Justificación:** Ejecutarlos $N$ veces tiene el mismo efecto en el estado que 1 vez. POST no lo es. |
| **`cloze_deletion`** | ⚡ **Local** | Aplicar | Memoria muscular para sintaxis o fórmulas. | **Q:** Comando para deshacer commit manteniendo cambios en stage: `git reset [___] HEAD~1`.<br>**Solución:** `--soft`.<br>**Justificación:** `--soft` retrocede HEAD pero deja archivos en stage; `--hard` los destruiría. |
| **`output_prediction`** | ⚡ **Local** | Aplicar | Forzar ejecución mental (compilador humano). | **Contexto:** `setTimeout(()=>console.log(2),0); Promise.resolve().then(()=>console.log(3));`<br>**Q:** Output exacto en consola.<br>**Solución:** `3, 2`.<br>**Justificación:** Promesas van a Microtask Queue (prioridad), `setTimeout` a Macrotask Queue. |
| **`ordering`** | ⚡ **Local** | Analizar | Entender causalidad y ciclos de vida. | **Q:** Ordena el "Three-way handshake" de TCP.<br>**Solución:** 1) SYN, 2) SYN-ACK, 3) ACK.<br>**Justificación:** Romper el orden impide sincronizar números de secuencia para una conexión fiable. |
| **`anomaly_detection`** | 🧠 **IA** | Analizar | Detectar fallos lógicos ocultos que sí compilan. | **Contexto:** `for i:=0; i<5; i++ { go func() { fmt.Print(i) }() }` (Golang)<br>**Q:** ¿Qué anomalía tiene y qué imprime?<br>**Criterio IA:** El usuario debe identificar la "captura de variable en closure". Imprime `55555`. |
| **`optimization`** | 🧠 **IA** | Evaluar | Refactorizar hacia la eficiencia matemática. | **Contexto:** Algoritmo de búsqueda de duplicados en $O(n^2)$.<br>**Q:** Refactoriza a complejidad $O(n)$.<br>**Criterio IA:** El usuario debe exigir el uso de un Hash Map (Diccionario) para registrar frecuencias en una sola pasada. |
| **`case_analysis`** | 🧠 **IA** | Crear | Diseño de sistemas y evaluación de *trade-offs*. | **Contexto:** DB relacional colapsa por lecturas en Black Friday.<br>**Q:** Arquitectura de mitigación en 1 hora.<br>**Criterio IA:** Validar propuesta de Capa de Caché en memoria (Redis/Memcached), justificando tolerancia a consistencia eventual. |
| **`feynman_synthesis`** | 🧠 **IA** | Maestría | Transferencia de conocimiento sin jerga. | **Q:** Explica Hashing vs Encriptación a un Junior.<br>**Criterio IA:** Hashing es unidireccional (integridad), Encriptación bidireccional (protección). Penalizar severamente el uso de jerga matemática abstracta. |
